depth_change(id)

Description

Temporarily changes the current depth buffer to another surface's depth buffer. The other surface must have the same width and height as the current surface.

Arguments

This function requires that you pass in 1 argument:
  1. id - Real. The ID of the surface whose depth buffer you want to change to. The surface must exist, and it must have the same dimensions as the current surface. If you pass in -1 for this argument, you will change to the screen's depth buffer.

Return Value

Real. This function returns 1 (true) if it succeeds, or 0 (false) if it fails.

Example

// In this example, the "surface" variable holds the ID of a surface that was created earlier.

// Switch to the surface
surface_set_target(surface);
draw_clear(c_black);

// Use the screen's depth buffer
depth_change(-1);

// DRAWING CODE GOES HERE!

// Reset the render target
surface_reset_target();
