depth_write_enable(enable)

Description

Enables or disables writing to the depth buffer. When writing is disabled, hidden surface removal will still be used, but the depth buffer will not change even when you draw objects. Depth buffer writing is enabled by default.

Arguments

This function requires that you pass in 1 argument:
  1. enable - Real. Pass in a non-zero value (true) to enable depth writing, or pass in zero (false) to disable it.

Return Value

This function does not return any meaningful value.

Example

// Disable depth writing
depth_write_enable(false);

// Draw a blue cube
draw_set_color(c_blue);
d3d_draw_block(-16, -16, -16, 16, 16, 16, -1, 0, 0);

// Re-enable depth writing
depth_write_enable(true);
