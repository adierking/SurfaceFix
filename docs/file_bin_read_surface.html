file_bin_read_surface(file, id)

Description

Reads a surface from an already opened binary file. The destination surface must exist and have the same width and height as the one read from the binary file. The surface's old contents will be discarded.

Arguments

This function requires that you pass in 2 arguments:
  1. file - Real. The ID of the binary file to read the surface from. It must have been opened for read access.
  2. id - Real. The ID of the surface to read from the file. It must exist, and it must have the same dimensions as the one read from the file.

Return Value

Real. This function returns 1 (true) if it succeeds, or 0 (false) if it fails.

Example

// In this example, the variable "surface" holds the ID of a surface that was created earlier.

// Open a binary file for reading
file = file_bin_open("surface.dat", 0);

// Read the surface from it
file_bin_read_surface(file, surface);

// Close the file
file_bin_close(file);
