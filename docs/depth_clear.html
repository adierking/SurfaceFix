depth_clear()

Description

Clears the depth buffer of the surface that you are currently drawing to.

As of Surface Fix version 1.2, this function has become obsolete and is provided only for compatibility purposes. It has been merged into draw_clear() and draw_clear_alpha(), making it no longer necessary to use.

Arguments

This function does not accept any arguments.

Return Value

This function does not return any meaningful value.

Example

depth_clear();
