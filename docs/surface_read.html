surface_read(id, str)

Description

Takes a string returned by surface_write() and copies its data into a surface. The data format is auto-detected. The surface must already exist, and its old contents will be discarded.

Arguments

This function requires that you pass in 2 arguments:
  1. id - Real. The ID of the surface to copy the data into. It must already exist, and it must have the same width and height as the surface whose data is in the string.
  2. str - String. The string containing the surface data. Its format is auto-detected.

Return Value

Real. This function returns 1 (true) if it succeeds, or 0 (false) if it fails.

Example

// This example demonstrates how to preserve a surface across a display_reset() call.
// The variable "surface" holds the ID of a 640x480 surface that was created earlier.

var contents;

// Grab the contents of the surface
contents = surface_write(surface, df_binary);

// Reset the display
display_reset();

// The display reset destroyed the surface, so re-create the surface
surface = surface_create(640, 480);

// Restore the surface's contents
surface_read(surface, contents);
