surface_get_target()

Description

Returns the ID of the surface that you are currently drawing to.

Arguments

This function does not accept any arguments.

Return Value

Real. This function returns the ID of the surface that you are currently drawing to. If you are currently drawing to the screen, -1 is returned.

Example

// In this example, "my_surface" is a variable holding the ID of a surface that was created earlier.

current_surface = surface_get_target();
if (current_surface == my_surface)
{
    draw_text(8, 8, "You are drawing to my_surface.");
}
else if (current_surface == -1)
{
    draw_text(8, 8, "You are drawing to the screen.");
}
