fix_surfaces([dllpath])

Description

Initializes the Surface Fix DLL. You must call this only once, when your game initializes.

Arguments

This function accepts one optional argument:
  1. dllpath - String. The path to SurfaceFix.dll. If this argument is not specified, it is assumed to be in the current directory.

Return Value

This function does not return any meaningful value.

Example

fix_surfaces();
